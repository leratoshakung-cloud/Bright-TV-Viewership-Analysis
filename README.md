 # 📺Bright TV Viewership Analysis

 # 📋 Overview

This repository contains the analysis and strategic recommendations for Bright TV’s user base.
BrightTV’s CEO has set an objective to grow the company’s subscription base within the current financial year. He needs the insights to assist the Customer Value Management team to reach this objective. The data is from the period: 01 January - 01 April 2016

#  🎯Objective

The main objective is to the assist the CEO to grow the company subscription base, so the analysis covers the following:

• Insights on user and usage trends of BrightTV. 

• What type of factors influence consumption

• Content recommendation to increase consumption on the days with low 
consumption? 

#  💻Analytical Tools Applied

•**Data processing**: The SQL queries was completed in Databricks platform, then dowloaded the processed data into Microsoft excel for analysis

•**Project Planning & Archetecture**: I used Miro to brainstorm and to create a flow diagram. I also used Canva to build the project timelines

•**Data Visualization**: To visualise i used Microsoft Excel to built the analysis using pivot tables and charts, then i used Google Data Studio to build a dashboard.

•**Presentation**: Final analysis was presented on Microsoft Powerpoint


# 💡Key insights

1️. Growth trend:	Strong month-over-month growth observed from Jan→Feb (+35.5%) and Feb→Mar (+61.6%), indicating successful user acquisition strategies

2️. Demographics:	Male users dominate (87.6%), revealing significant opportunity to grow female audience through targeted content and marketing

3️. Geographic concentration:	Gauteng (36.5%) and Western Cape (18.5%) account for 55% of users - consider expansion strategies for underserved provinces

4️. Content Preference:	Sport content leads (40%) driven by ICC Cricket World Cup and SuperSport - leverage sports events for user engagement

5️. Prime time:	Afternoon (24.6%) and Late Night (19.1%) slots show highest engagement - optimize ad scheduling and content releases

6️. Age Group:	Adults (91%) form the core user base - opportunity to develop dedicated content for teenagers (4.4%) and kids (1%)
